---
title: Review  
description: Representa un modelo de Reseña, diseñado para almacenar información detallada sobre el comentario de un usuario que ya se hospedó en una propiedad.
---

### Modelo

```json
{
    "_id" : ObjectId("679eb0ba5d663ab7ad2b7318"),
    "user_id" : ObjectId("679eb2e10674ac227ab940f9"),
    "booking_id" : ObjectId("679eb2e10674ac227ab940f9"),
    "score" : 4.5,
    "comment" : "Excelente lugar para descansar y disfrutar de la naturaleza.",
    "date" : ISODate("2022-01-15T00:00:00Z")
}
```

### Campos

| Campo               | Tipo         | Descripción                                                                 |
|---------------------|--------------|-----------------------------------------------------------------------------|
| `_id`               | `ObjectId`   | Identificador único del documento de la reseña.                             |
| `user_id`           | `ObjectId`   | Identificador del usuario que escribió la reseña.                           |
| `booking_id`        | `ObjectId`   | Identificador de la reserva asociada a la reseña.                           |
| `score`             | `Number`     | Calificación otorgada por el usuario (valor entre 1 y 5).                   |
| `comment`           | `String`     | Comentario del usuario sobre la propiedad.                                  |
| `date`              | `Date`       | Fecha en que se dejó la reseña.                                             |
