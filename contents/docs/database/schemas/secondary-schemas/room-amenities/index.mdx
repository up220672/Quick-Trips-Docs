---
title: Room Amenities Schema
description: Representa un esquema de Amenidades, diseñado para almacenar información detallada sobre los servicios y características de una habitación en una propiedad.
---

Incluye información sobre la disponibilidad de televisión, clóset, aire acondicionado, calefacción, ventilador, escritorio, espejo, lámparas, wifi, sofá, cortinas, mini refrigerador, cobijas extra, almohadas extra, alfombra, estante para libros, cajones, tomas de corriente, sistema de sonido y reloj despertador.

### Schema

const RoomAmenitiesSchema = new mongoose.Schema({
  tv: { 
    type: Boolean, 
    required: false,

### Campos

| Campo              | Tipo         | Descripción                                                                 |
|--------------------|--------------|-----------------------------------------------------------------------------|
| `_id`              | `ObjectId`   | Identificador único del documento que representa las amenidades de la habitación. |
| `tv`               | `Boolean`    | Indica si hay una televisión disponible. Valor por defecto: `false`.       |
| `closet`           | `Boolean`    | Determina si la habitación incluye un clóset. Valor por defecto: `false`.   |
| `air_conditioner`  | `Boolean`    | Verifica si hay aire acondicionado. Valor por defecto: `false`.             |
| `heater`           | `Boolean`    | Indica si hay calefacción. Valor por defecto: `false`.                      |
| `fan`              | `Boolean`    | Indica si hay ventilador. Valor por defecto: `false`.                       |
| `desk`             | `Boolean`    | Confirma si hay un escritorio. Valor por defecto: `false`.                  |
| `mirror`           | `Boolean`    | Indica si hay un espejo. Valor por defecto: `false`.                        |
| `lamps`            | `Number`     | Número de lámparas disponibles. Valor por defecto: `0`.                     |
| `wifi`             | `Boolean`    | Indica si hay conexión a internet inalámbrica. Valor por defecto: `false`.  |
| `sofa`             | `Number`     | Número de sofás disponibles. Valor por defecto: `0`.                        |
| `curtains`         | `Boolean`    | Confirma si hay cortinas estándar. Valor por defecto: `false`.              |
| `blackout_curtains`| `Boolean`    | Indica si hay cortinas blackout. Valor por defecto: `false`.                |
| `mini_fridge`      | `Boolean`    | Verifica si hay un mini refrigerador. Valor por defecto: `false`.           |
| `extra_blankets`   | `Number`     | Número de cobijas extra disponibles. Valor por defecto: `0`.                |
| `extra_pillows`    | `Number`     | Número de almohadas extra disponibles. Valor por defecto: `0`.              |
| `rug`              | `Boolean`    | Indica si hay una alfombra. Valor por defecto: `false`.                     |
| `bookshelf`        | `Boolean`    | Indica si hay un estante para libros. Valor por defecto: `false`.           |
| `drawers`          | `Number`     | Cantidad de cajones disponibles. Valor por defecto: `0`.                    |
| `power_outlets`    | `Number`     | Número de tomas de corriente disponibles. Valor por defecto: `0`.           |
| `sound_system`     | `Boolean`    | Indica si hay un sistema de sonido. Valor por defecto: `false`.             |
| `alarm_clock`      | `Boolean`    | Verifica si hay un reloj despertador. Valor por defecto: `false`.           |

**Notas adicionales**

- **`versionKey: false`**: Este parámetro en el esquema de Mongoose desactiva la inclusión del campo `__v` que Mongoose añade por defecto para controlar la versión del documento.
- **Valores por defecto**: Todos los campos booleanos tienen un valor por defecto de `false`, y los campos numéricos tienen un valor por defecto de `0`. Esto asegura que, si no se proporciona un valor, el campo se inicialice con un valor predeterminado seguro.
