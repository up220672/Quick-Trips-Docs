---
title: Video Schema
description: Representa un esquema de Video, almacena la información de acceso y la metadata básica de un archivo de video.
---

### Schema

```javascript
const mongoose = require('mongoose');

const VideoSchema = new mongoose.Schema({
  url: { 
    type: String, 
    required: false,
    default: null,
    trim: true 
  },
  title: { 
    type: String, 
    required: false,
    default: null,
    trim: true 
  },
  description: { 
    type: String, 
    required: false,
    default: null,
    trim: true 
  },
  duration: { 
    type: Number, 
    required: false,
    default: null,
    min: 0 
  },
  width: { 
    type: Number, 
    required: false,
    default: null,
    min: 0 
  },
  height: { 
    type: Number, 
    required: false,
    default: null,
    min: 0 
  },
  format: { 
    type: String, 
    required: false,
    default: null,
    trim: true 
  },
  size_mb: { 
    type: Number, 
    required: false,
    default: null,
    min: 0 
  },
  thumbnail_url: { 
    type: String, 
    required: false,
    default: null,
    trim: true 
  }
}, { versionKey: false });

const Video = mongoose.model('Video', VideoSchema);

module.exports = { Video, VideoSchema };
```

### Campos

| Campo              | Tipo         | Descripción                                                                 |
|--------------------|--------------|-----------------------------------------------------------------------------|
| `_id`              | `ObjectId`   | Identificador único del documento del video.                               |
| `url`              | `String`     | URL del video almacenado en un servicio de alojamiento de archivos. Si no se proporciona, el valor por defecto es `null`. Se eliminan los espacios en blanco al principio y al final de la URL. |
| `title`            | `String`     | Título del video. Si no se proporciona, el valor por defecto es `null`. Se eliminan los espacios en blanco al principio y al final del título. |
| `description`      | `String`     | Descripción del video. Si no se proporciona, el valor por defecto es `null`. Se eliminan los espacios en blanco al principio y al final de la descripción. |
| `duration`         | `Number`     | Duración del video en segundos. Si no se proporciona, el valor por defecto es `null`. El valor mínimo permitido es `0`. |
| `width`            | `Number`     | Ancho del video en píxeles. Si no se proporciona, el valor por defecto es `null`. El valor mínimo permitido es `0`. |
| `height`           | `Number`     | Alto del video en píxeles. Si no se proporciona, el valor por defecto es `null`. El valor mínimo permitido es `0`. |
| `format`           | `String`     | Formato del video (ej: "mp4", "webm"). Si no se proporciona, el valor por defecto es `null`. Se eliminan los espacios en blanco al principio y al final del formato. |
| `size_mb`          | `Number`     | Tamaño del archivo del video en megabytes (MB). Si no se proporciona, el valor por defecto es `null`. El valor mínimo permitido es `0`. |
| `thumbnail_url`    | `String`     | URL de la miniatura del video. Si no se proporciona, el valor por defecto es `null`. Se eliminan los espacios en blanco al principio y al final de la URL. |

**Notas adicionales**

- **`versionKey: false`**: Este parámetro en el esquema de Mongoose desactiva la inclusión del campo `__v` que Mongoose añade por defecto para controlar la versión del documento.
- **`trim: true`**: Este parámetro en los campos `url`, `title`, `description`, `format` y `thumbnail_url` asegura que cualquier espacio en blanco al principio o al final del texto sea eliminado antes de guardarlo en la base de datos.
- **`min: 0`**: Este parámetro en los campos `duration`, `width`, `height` y `size_mb` asegura que los valores no sean negativos.
