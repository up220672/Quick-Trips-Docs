---
title: Beds  
description: Representa un esquema para almacenar información detallada sobre la cantidad de camas en una habitación.  
---

### Schema

```json
{
    "_id": ObjectId("679eb0ba5d663ab7ad2b7319"),
    "king_size": 0,
    "queen_size": 0,
    "single": 1,
    "sofa_bed": 1,
    "bunk_beds": 0,
    "cradle": 0
}
```

### Campos

| Campo       | Tipo         | Descripción                                                             |
|-------------|--------------|-------------------------------------------------------------------------|
| `_id`       | `ObjectId`   | Identificador único del documento que representa las camas de la habitación. |
| `king_size` | `Number`     | Número de camas tamaño King Size disponibles en la habitación.         |
| `queen_size`| `Number`     | Número de camas tamaño Queen Size / Matrimoniales disponibles en la habitación.        |
| `single`    | `Number`     | Número de camas individuales disponibles en la habitación.             |
| `sofa_bed`  | `Number`     | Número de sofás cama disponibles en la habitación.                     |
| `bunk_beds` | `Number`     | Número de literas disponibles en la habitación.                        |
| `cradle`    | `Number`     | Número de cunas disponibles en la habitación.                         |
