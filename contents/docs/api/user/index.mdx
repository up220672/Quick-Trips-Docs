## Crear un nuevo usuario
**( CREATE | POST )**

`http://localhost:3000/api/user`

Crea un nuevo usuario en la base de datos.

- **URL**: `/user`

- **Parámetros**:
  - **Body** (JSON):
    ```json
    {
        "username": "juanperez",
        "first_name": "Juan",
        "last_name": "Pérez",
        "email": "juan@example.com",
        "phone": "+521234567890",
        "password_hash": "68efa2a44f66ee921ea728ece5371940d41ac3b5",
        "date_of_birth": ISODate("2023-10-05T14:30:00Z"),
        "language": "es",
        "address": {
            "street": "Calle Falsa 123",
            "exterior_number": "123",
            "interior_number": "A",
            "neighborhood": "Narvarte",
            "city": "CDMX",
            "state": "CDMX",
            "postal_code": "03020",
            "country": "México",
            "references": "Entre Calles Verdadera y Ficticia, cerca del parque",
        },
        "currency": "MXN"
    }
    ```



## Obtener un usuario
**( READ | GET )** 

`http://localhost:3000/api/user/67a162e22332616556e99b70`

Obtiene la información de un usuario en la base de datos.

- **URL**: `/user/:id`

- **Parámetros**:
  - **Path**:
    - `id` (String): Identificador único del usuario.


## Actualizar un usuario
**( UPDATE | PUT )**

`http://localhost:3000/api/user/67a162e22332616556e99b70`

Cambia solo los valores que recibe

- **URL**: `/user/:id`

- **Parámetros**:
  - **Path**:
    - `id` (String): Identificador único del usuario.
  - **Body** (JSON):
    ```json
    {
      "email": "new_mail@gmail.com",
    }
    ```

## Eliminar un usuario
**( DELETE | DELETE )**

`http://localhost:3000/api/user/67a162e22332616556e99b70`

Actualiza el campo `is_active` a `false`

- **URL**: `/user/:id`

- **Parámetros**:
  - **Path**:
    - `id` (String): Identificador único del usuario.


## Obtener todos los usuarios
**( READ | GET )**

`http://localhost:3000/api/users`

Obtiene la información de todos los usuarios en la base de datos.

- **URL**: `/users`

- **Parámetros** : Ninguno
