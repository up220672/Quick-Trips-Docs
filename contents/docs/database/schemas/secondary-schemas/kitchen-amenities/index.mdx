---
title: Kitchen Amenities
description: Representa un esquema de Amenidades de Cocina, diseñado para almacenar información detallada sobre los servicios y características de una cocina en una propiedad.
---

### Schema

```json
{
    "_id": ObjectId("679eb0ba5d663ab7ad2b7321"),
    "refrigerator": true,
    "stove": true,
    "oven": true,
    "microwave": true,
    "dishwasher": false,
    "sink": true,
    "counter_space": true,
    "coffee_maker": true,
    "toaster": true,
    "kettle": true,
    "blender": false,
    "cutlery": 4,
    "plates": 4,
    "bowls": 4,
    "glasses": 4,
```

### Campos

| Campo               | Tipo         | Descripción                                                             |
|---------------------|--------------|-------------------------------------------------------------------------|
| `_id`               | `ObjectId`   | Identificador único del documento que representa las amenidades de la cocina. |
| `refrigerator`      | `Boolean`    | Indica si hay un refrigerador disponible.                              |
| `stove`             | `Boolean`    | Verifica si hay una estufa disponible.                                  |
| `oven`              | `Boolean`    | Indica si hay un horno disponible.                                      |
| `microwave`         | `Boolean`    | Indica si hay un microondas disponible.                                 |
| `dishwasher`        | `Boolean`    | Verifica si hay un lavaplatos disponible.                               |
| `sink`              | `Boolean`    | Indica si hay un fregadero en la cocina.                                |
| `counter_space`     | `Boolean`    | Indica si hay suficiente espacio en las superficies para trabajar.      |
| `coffee_maker`      | `Boolean`    | Indica si hay una cafetera disponible.                                  |
| `toaster`           | `Boolean`    | Verifica si hay una tostadora disponible.                               |
| `kettle`            | `Boolean`    | Indica si hay una tetera eléctrica disponible.                          |
| `blender`           | `Boolean`    | Verifica si hay una licuadora disponible.                               |
| `cutlery`           | `Number`     | Cantidad de utensilios de cocina (tenedores, cuchillos, cucharas).      |
| `plates`            | `Number`     | Cantidad de platos disponibles en la cocina.                           |
| `bowls`             | `Number`     | Cantidad de tazones disponibles.                                       |
| `glasses`           | `Number`     | Cantidad de vasos disponibles.                                         |
| `pots_and_pans`     | `Boolean`    | Verifica si hay ollas y sartenes disponibles.                           |
| `cooking_utensils`  | `Boolean`    | Indica si hay utensilios de cocina como cucharas de madera, espátulas, etc. |
| `spices`            | `Boolean`    | Indica si se ofrecen especias básicas (comino, orégano, etc.).         |
| `salt`              | `Boolean`    | Verifica si hay sal disponible en la cocina.                           |
| `pepper`            | `Boolean`    | Indica si hay pimienta disponible en la cocina.                        |
| `trash_bin`         | `Boolean`    | Indica si hay un bote de basura en la cocina.                           |
| `recycling_bin`     | `Boolean`    | Verifica si hay un bote de reciclaje.                                  |
| `paper_towels`      | `Boolean`    | Indica si hay toallas de papel disponibles.                            |
| `cleaning_supplies` | `Boolean`    | Indica si se ofrecen productos de limpieza (detergente, esponjas, etc.). |
| `power_outlets`     | `Number`     | Cantidad de tomas de corriente disponibles en la cocina.                |
