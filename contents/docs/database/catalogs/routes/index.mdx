---
title: Rutas  
description: Colección de las rutas de la aplicación QuickTrips.
---

Está colección esta diseñada para centrar las rutas del sistema y poder mandar notificaciones que naveguen a las pantallas correctas sin importar que el sistema cambie

```json
{
    "booking_route" : "/booking",
}
```

### Campos

| Campo               | Tipo         | Descripción                                                                 |
|---------------------|--------------|-----------------------------------------------------------------------------|
| `booking_route`     | `String`     | Ruta de la pantalla de reservas.                                            |
