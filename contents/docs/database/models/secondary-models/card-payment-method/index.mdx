---
title: Card Payment Method  
description: Representa un modelo de Método de Pago con Tarjeta, diseñado para almacenar información detallada sobre los métodos de pago con tarjeta de crédito o débito de los usuarios. Solo se almacenan datos no sensibles para garantizar la seguridad.
---

### Modelo

```json
{
    "_id": ObjectId("679ecc761a3e63fcc4a6a3ac"),
    "name" : "John Doe",
    "last4": "1111",
    "card_type": "Visa",
    "expiration_date": "12/25",
    "address": address()
}
```

### Campos

| Campo               | Tipo         | Descripción                                                                       |
|---------------------|--------------|-----------------------------------------------------------------------------------|
| `_id`               | `ObjectId`   | Identificador único del documento de método de pago.                              |
| `name`              | `String`     | Nombre del titular de la tarjeta (máscara para seguridad).                        |
| `last4`             | `String`     | Últimos 4 dígitos de la tarjeta (máscara para seguridad).                         |
| `card_type`         | `String`     | Tipo de tarjeta (ej: Visa, Mastercard, American Express).                         |
| `expiration_date`   | `String`     | Fecha de expiración de la tarjeta (MM/YY).                                        |
| `address`           | `Address`    | Dirección de facturación asociada a la tarjeta. Ver [Address](/docs/database/models/secondary-models/address). |
