---
title: Property Type
description: Colección tipo catálogo que almacena los tipos de propiedades disponibles en QuickTrips.
---

### Modelo

```json
{
    "0": "Privado",
    "1": "Compartido",
}
```

### Campos

| **Clave** | **Tipo de Baño**  | **Descripción**                                                                 |
|-----------|------------------------|---------------------------------------------------------------------------------|
| `0`       | Privado          | Baño privado, solo para uso de los huéspedes de la reservación.   |
| `1`       | Compartido           | Baño compartido con otros huéspedes o residentes de la propiedad. |
