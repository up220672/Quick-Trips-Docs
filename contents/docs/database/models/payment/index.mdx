---
title: Payment  
description: Representa un modelo de Pago, diseñado para almacenar información detallada sobre las transacciones realizadas por los usuarios en la plataforma.
---

### Modelo

```json
{
    "_id": ObjectId("679eb0ba5d663ab7ad2b7318"),
    "payment_id": ObjectId("679eb2e10674ac227ab940f9"),
    "booking_id": ObjectId("679eb2e10674ac227ab940f9"),
    "user_id": ObjectId("679eb2e10674ac227ab940f9"),
    "property_id": ObjectId("679eb2e10674ac227ab940f9"),
    "host_id": ObjectId("679eb2e10674ac227ab940f9"),
    "amount_details": AmountDetails(),
    "card" : CardPaymentMethod(),
    "current_status": 0,
    "created_at": ISODate("2021-09-01T00:00:00Z"),
    "confirmed_at": ISODate("2021-09-01T00:00:00Z")
}
```

### Campos

| Campo               | Tipo         | Descripción                                                                 |
|---------------------|--------------|-----------------------------------------------------------------------------|
| `_id`               | `ObjectId`   | Identificador único del documento de pago.                                  |
| `payment_id`        | `ObjectId`   | Identificador único de la transacción de pago.                              |
| `booking_id`        | `ObjectId`   | Identificador de la reserva asociada al pago.                               |
| `user_id`           | `ObjectId`   | Identificador del usuario que realizó el pago.                              |
| `property_id`       | `ObjectId`   | Identificador de la propiedad asociada al pago.                             |
| `host_id`           | `ObjectId`   | Identificador del host que recibe el pago.                                  |
| `amount_details`    | `Object`     | Detalles del monto asociado al pago. Ver [Amount Details](/docs/database/models/secondary-models/amount-details). |
| `card`              | `Object`     | Método de pago con tarjeta asociado al pago. Ver [Card Payment Method](/docs/database/models/secondary-models/card-payment-method). |
| `current_status`    | `Number`     | Estado actual del pago. Ver [Card Payment Method](/docs/database/catalogs/payment-status-type). |
| `created_at`        | `Date`       | Fecha y hora de creación del pago.                                          |
| `confirmed_at`      | `Date`       | Fecha y hora de confirmación del pago.                                      |
