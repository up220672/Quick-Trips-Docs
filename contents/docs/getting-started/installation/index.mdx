---
title: Instalación
description: Guía de instalación de QuickTrips.
---

Configurar **QuickTrips** es un proceso sencillo. Puedes clonar el repositorio o configurarlo manualmente siguiendo los pasos a continuación. Asegúrate de tener instalada la versión 16 o superior de [Node.js](https://nodejs.org/) y [MongoDB](https://www.mongodb.com/).

## Clonar el Repositorio
Para comenzar, clona el repositorio de **QuickTrips** desde GitHub.

<Stepper>
  <StepperItem title="Paso 1: Clonar el Repositorio de QuickTrips">
    Clona el repositorio ejecutando el siguiente comando:

    ```plaintext
    git clone https://github.com/tuusuario/quicktrips.git
    ```
  </StepperItem>

  <StepperItem title="Paso 2: Acceder al Directorio del Proyecto">
    Luego de clonar, navega dentro del directorio del proyecto:

    ```plaintext
    cd quicktrips
    ```
  </StepperItem>

  <StepperItem title="Paso 3: Instalar Dependencias Requeridas">
    Instala todas las dependencias necesarias con npm:

    ```plaintext
    npm install
    ```
  </StepperItem>

  <StepperItem title="Paso 4: Configurar Variables de Entorno">
    Crea un archivo `.env` en la raíz del proyecto y añade las variables de entorno necesarias:

    ```plaintext
    MONGO_URI=your_mongodb_connection_string
    JWT_SECRET=your_secret_key
    ```
  </StepperItem>

  <StepperItem title="Paso 5: Iniciar el Servidor de Desarrollo">
    Finalmente, ejecuta el siguiente comando para iniciar el servidor:

    ```plaintext
    npm run dev
    ```
  </StepperItem>
</Stepper>

## Configuración Rápida con npx

Para una instalación más rápida, usa el siguiente comando `npx`:

```plaintext
npx create-quicktrips <directorio-del-proyecto>
```

**Salida esperada:**

```plaintext
Creando un nuevo proyecto QuickTrips en /ruta/del/proyecto...
Clonando repositorio...
QuickTrips se ha configurado correctamente en /ruta/del/proyecto.

Pasos siguientes:
1. Navegar al directorio del proyecto:
   cd <directorio-del-proyecto>
2. Instalar dependencias:
   npm install
3. Iniciar el servidor de desarrollo:
   npm run dev
```

Con este proceso, tendrás **QuickTrips** listo para empezar a gestionar alojamientos y reservas. 🚀
