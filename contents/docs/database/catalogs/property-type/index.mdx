### Modelo

```json
{
    "0": "Casa completa",
    "1": "Departamento",
    "2": "Cabaña",
    "3": "Habitación privada",
    "4": "Casa de campo",
    "5": "Villa",
    "6": "Loft",
    "7": "Estudio",
}
```

### Campos

| **Clave** | **Tipo de Vivienda**  | **Descripción**                                                                 |
|-----------|------------------------|---------------------------------------------------------------------------------|
| `0`       | Casa completa          | Vivienda independiente donde los huéspedes tienen acceso a toda la propiedad.   |
| `1`       | Departamento           | Vivienda privada en un edificio con varias unidades, generalmente de varios pisos. |
| `2`       | Cabaña                 | Vivienda pequeña generalmente ubicada en áreas rurales o naturales, como bosques o montañas. |
| `3`       | Habitación privada     | Espacio privado dentro de una vivienda, donde el huésped tiene su propio acceso, pero comparte algunas áreas comunes con otros huéspedes. |
| `4`       | Casa de campo          | Vivienda rural, generalmente aislada y rodeada de naturaleza, ideal para escapar del ajetreo de la ciudad. |
| `5`       | Villa                  | Vivienda lujosa, a menudo ubicada en áreas turísticas exclusivas, con gran espacio y comodidades. |
| `6`       | Loft                   | Vivienda de estilo industrial o moderno, usualmente de espacio abierto y de concepto único. |
| `7`       | Estudio                | Pequeña unidad independiente, típicamente de una sola habitación que combina sala, comedor y dormitorio. |
